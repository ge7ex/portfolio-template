Summary
- V40 keeps the V39 PrintDesigner / portrait PDF / pagination fixes.
- Mobile option B is added: desktop navbar stays unchanged, but mobile/tablet narrow view collapses the top controls into one branch menu button.
- Mobile branch menu includes Language, Portfolio, Resume, Theme, PDF, PPTX, and Edit actions.
- Mobile scrollytelling/experience layout is changed to a touch-friendly horizontal gallery and no longer depends on desktop scroll-height staging.
- Mobile hero/header spacing is tightened so the top area is less crowded and has less empty vertical space.
- Print PDF Landscape remains hidden/disabled. Portrait PDF remains the intended path.

Upload / Git usage
1. Checkout the working branch only:
   git checkout gpt/portfolio-working
   git pull origin gpt/portfolio-working

2. Replace the local AI-Portfolio-Toolkit folder with this package version.

3. Review before commit:
   git status
   git diff -- AI-Portfolio-Toolkit/index.html
   git diff -- AI-Portfolio-Toolkit/css/style.css
   git diff -- AI-Portfolio-Toolkit/js/components/PrintDesigner.js
   git diff -- AI-Portfolio-Toolkit/js/storage.js

4. Commit to working branch only:
   git add AI-Portfolio-Toolkit
   git commit -m "V40 add mobile branch controls and responsive scrollytelling"
   git push origin gpt/portfolio-working

QA checklist before merging to main
- Open on desktop/laptop: top nav should remain unchanged.
- Open on mobile width: top nav should show one compact menu/branch button.
- Tap the mobile branch button: Language, Portfolio, Resume, Theme, PDF, PPTX, and Edit should appear.
- Switch Portfolio/Resume from mobile branch and confirm the active button updates.
- Open Theme from mobile branch and confirm the existing theme wheel still opens.
- Scroll experience/projects on mobile and confirm images are touch-scrollable without a large blank scrollytelling gap.
- Open PDF Layout Studio and confirm real data appears, not placeholder data.
- Print PDF as A4 portrait.
- Do not test or expose PDF landscape; landscape PDF is intentionally disabled.
