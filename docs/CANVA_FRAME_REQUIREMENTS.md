# Canva Frame Requirements

The Canva brief must ask Canva to create replaceable picture frames, not decorative boxes.

Required frame labels:

- PIC_FRAME_01 — HERO PORTRAIT — 4:5
- PIC_FRAME_02 — FEATURED PROJECT — 16:9
- PIC_FRAME_03 — PROJECT GALLERY — 1:1
- PIC_FRAME_04 — PROOF / CERTIFICATION / ARTICLE — 4:3

Each frame must be visible, labeled, and replaceable by drag-and-drop in Canva.
