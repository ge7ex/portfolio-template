# AI Portfolio Toolkit v51 — Locked 9-Panel Grid + Human Theme Gradients

## Changes

- Reworked the background interaction into a true locked 3x3 / 9-panel grid.
- The grid is anchored to the viewport/background with visual axis and bolt cues.
- Panels flip based on mouse movement direction and hover position.
- Removed the old feel of page/book folding and whole-grid drifting.
- Refined all theme gradients to feel more natural, less synthetic, and more premium.
- Preserved existing storage, export, scrollytelling, quick-edit and theme logic.

- `css/style.css`

## Test checklist

1. Open `index.html`.
2. Move the mouse across the page. The 9 panels should stay locked to the background while flipping with mouse direction.
3. Switch all themes and verify the background gradient remains premium and readable.
4. Test portfolio desktop and mobile. The grid is hidden on mobile/reduced-motion/print.
